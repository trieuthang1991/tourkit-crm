## 3. GOTCHA quan trọng (đọc kỹ)
- **LUÔN kill API (:5075) trước `dotnet build`/`dotnet test`** — DLL lock khiến test fail IM LẶNG (xanh giả). Xem memory `kill-api-before-dotnet-test`.
- **Seeder** (`src/TourKit.Api/DevData/DemoDataSeeder.cs`): idempotent per-entity; có **early-return nếu OD_0001 tồn tại** (section 8–12 chỉ chạy fresh). Đổi seed data cũ phải **xoá row qua psql** (`docker exec tourkit-postgres psql -U tourkit -d tourkit_dev -c 'DELETE FROM "X";'`) — API DELETE hay trả 000 không tin cậy. Nếu còn row cũ + seed lại → **crash duplicate key chặn startup**.
- **CA/IDE analyzers = ERROR** (IDE0005 using thừa, CA1859 kiểu param). Build sẽ fail — sửa ngay.
- **jsonb value object**: phải `PropertyNameCaseInsensitive + CamelCase` (seed viết camelCase). Xem `FlightItinerary`.
- **antd Table**: field tên `children` đụng nested rows → set `childrenColumnName="__noNested"`.
